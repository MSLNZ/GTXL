GTX_ERR()¶

GTX_ERR(ref)¶

Display an error message

Parameters:ref – reference to a cell that is indicating an error

If an error occurs during execution of GTX(), indicated by #VALUE, an error message may be registered by GTXL internally. That message can be displayed by GTX_ERR().
